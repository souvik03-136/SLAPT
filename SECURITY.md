# Security Policy

## Supported Versions

Currently supporting:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ✅ Yes             |
| < 0.1   | ❌ No              |

## Reporting a Vulnerability

Found a security issue? Please **DO NOT** open a public issue.

### Instead:

1. **Email me directly:** [your-email@example.com]
2. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Your suggested fix (if you have one)

### What happens next:

- I'll respond within 48 hours
- I'll work on a fix ASAP
- I'll credit you in the fix (unless you prefer to stay anonymous)
- Once fixed, I'll publish a security advisory

## Safe Usage

- Keep SLAPT updated to the latest version
- Don't run untrusted SLAPT code from random sources
- Be careful with file imports/exports
